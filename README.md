# AI File Worker

AI File Worker — веб-сервис, который выполняет работу с документами с помощью искусственного интеллекта.

## Цель проекта

Создать сервис, который позволяет пользователю загрузить документ, выбрать задачу и получить готовый результат за несколько секунд.

Примеры задач:

- Краткое содержание
- Анализ договора
- Поиск ошибок
- Сравнение документов
- Извлечение данных

## Стек

- Python
- FastAPI
- Jinja2
- Cursor
- Git

## Запуск

```bash
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload
```

После запуска:

http://127.0.0.1:8000
