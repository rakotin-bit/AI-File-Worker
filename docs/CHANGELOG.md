## v0.3.6.1 — Model Performance Tracking

### Added

- Добавлено отображение производительности AI моделей.
- Добавлено логирование latency каждого запроса.
- Добавлен вывод текущей статистики модели после успешного ответа.

### Improved

- Улучшена прозрачность работы Model Router.
- Логи теперь показывают:
  - выбранную модель;
  - время ответа;
  - количество успешных запросов;
  - количество ошибок;
  - среднее время ответа.

### Verified

Проверена работа:

- OpenRouter Provider;
- Model Router;
- Model Priority Manager;
- Model Stats;
- SQLite хранения статистики.

Пример результата:

Model:

tencent/hy3:free

Statistics:

- success_count: 7
- failure_count: 0
- avg_response_time: 5.56s

### Reason

Выбор AI модели должен учитывать не только доступность и успешность, но и пользовательский опыт через скорость получения результата.

### Status

✅ Работает
