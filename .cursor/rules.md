Проект: AI File Worker

Главная цель:

Максимально быстро вывести MVP на рынок.

Основные правила:

- Простота важнее красоты.
- Не писать лишний код.
- Один файл — одна ответственность.
- Использовать type hints.
- Писать понятные комментарии.
- Не создавать абстракции без необходимости.
- Каждый endpoint должен быть простым.
- Любое изменение должно быть минимальным.
